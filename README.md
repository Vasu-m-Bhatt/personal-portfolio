# personal-portfolio website
This project is a simple personal portfolio website created using HTML as part of the Week 1 assignment.
The purpose of this project is to learn the basic structure of a web page using HTML.
## Technologies Used
- HTML5

## Sections Included
- About Me
- Skills
- Contact Form

## Folder Structure
- index.html  
- images/ (contains profile image)

## Setup Instructions
1. Download or clone the repository
2. Open the index.html file in any web browser

## Concepts Learned
- HTML document structure
- Semantic HTML tags (header, nav, section, footer)
- Forms and input fields
- Internal navigation links
## Notes
This project was created for learning and practice purposes .
